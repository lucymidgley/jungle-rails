## Dependencies

* Rails 4.2 [Rails Guide](http://guides.rubyonrails.org/v4.2/)
* PostgreSQL 9.x
* Stripe

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run `bin/rails s -b 0.0.0.0` to start the developement server
